## 프로젝트 소개
- Weverse를 모작하여 제작한 사이트이며 Weverse에서 더 개발할 수 있는 부분이나 추가할 수 있는 기능들을 구현한 사이트

  ---
 ## 기술 스택
| 구분 | 사용 기술 |

| Backend | Java, Spring Boot, Python |
| Frontend | JSP, jQuery, HTML/CSS |
| DB | Oracle |
| 협업툴 | GitHub, Figma, Slack |
| 기타 | log4jdbc, JWT, WebSocket 등 |

## 주요 기능
- 로그인/회원가입 
- 아티스트 기반 커뮤니티 게시판 
- 아티스트 실시간 채팅방 
- 상품 등록 및 장바구니 기능
- 굿즈샵 및 결제
- 아티스트 추천 시스템
- 관리자 페이지
- 티켓팅 사이트
